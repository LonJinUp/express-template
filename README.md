# express-template

> 基于`express`的项目模版，数据库采用`mongodb`，可以在快读速搭建项目。

## 目录结构
```
express-template
├─ app.js
├─ config
│  └─ config.default.js
├─ controller
│  └─ testController.js
├─ middleware
│  └─ responseMiddleware
│     └─ index.js
├─ model
│  ├─ index.js
│  └─ testModel.js
├─ package.json
├─ router
│  ├─ index.js
│  └─ test.js
└─ utils
   └─ index.js
```
